# Clinica — Build Progress Journal

This is my learning journal for building Clinica. One entry per working day.
The point is not just to track tasks, but to learn **how to break a real
project into small, shippable chunks**. On future projects I can re-read this
to remember the rhythm.

How to use it:
- Before I start a day, I fill in **Planned**.
- As I work, I fill **Done**, **Learned**, **Blockers**, and **Next up**.
- Keep entries short and honest. "Done" should match reality, not the plan.

---

## Reusable daily template (copy this for each new day)

```
## Week X — Day N — <YYYY-MM-DD>

**Planned**
- [ ] task
- [ ] task

**Done**
- task (what actually happened)

**Learned**
- 1-3 takeaways in my own words

**Blockers / questions**
- anything that slowed me down or I want to revisit

**Next up**
- the first thing I'll do tomorrow
```
